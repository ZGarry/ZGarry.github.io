# 换三张

浏览器可玩的对门协作麻将小游戏。

- 开局可换三张。
- 一张牌会被指定为癞子，可作为任意牌参与胡牌。
- 我和队友互为对门，双方手牌可见。
- 任意一方完成 `4 * 3 + 2` 即队伍获胜。
- 游戏内置评分、预给牌、智能换牌、推荐出牌和验证局。

## 直接游玩

双击打开 `index.html`，或用任意静态服务托管本目录。

## 本地验证

```bash
node tests/core-tests.js
```

浏览器验收脚本需要 Playwright 和可用 Chrome：

```bash
node tests/browser-verify.js
```
